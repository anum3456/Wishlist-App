# Wishlist App

A simple and modern **React Wishlist Application** built using **Vite + Tailwind CSS**.  
This project allows users to add and remove items from their wishlist with a clean and responsive UI.

---

## 🚀 Features
- Add items to wishlist
- Remove items from wishlist
- Responsive and modern UI
- Fast performance using Vite
- Clean design with Tailwind CSS

---

## 🛠 Tech Stack
- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)

---

## 📦 Installation

Follow these steps to run the project locally:

### 1️⃣ Clone the repository
```bash
git clone https://github.com/anum3456/Wishlist-App.git

cd Wishlist-App

npm install

npm run dev
http://localhost:5173

```
